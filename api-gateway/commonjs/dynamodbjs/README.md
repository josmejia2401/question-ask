## common

Librería de utilitarios reutilizables para Python y Node.js.

### Estructura:
```
common/
├── python/
│   └── common/
│       ├── rabbitmq_utils.py
│       ├── event.py
│       └── logger.py
├── node/
│   └── common/
│       ├── rabbitmq.js
│       ├── event.js
│       └── logger.js
```

### Uso:
- Instalable localmente con `pip install -e ../common/python` o `npm install file:../common/node/common`
